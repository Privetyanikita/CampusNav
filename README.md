## 📖 О проекте

**CampusNav** — это удобное iOS-приложение, созданное для того, чтобы сделать жизнь студентов проще. От интерактивной карты кампусов до отслеживания домашних заданий и свежих новостей университета.

Приложение спроектировано в современном стиле (по гайдлайнам Apple), отличается плавными анимациями, быстрой версткой без Storyboard и удобной навигацией.

---

## ✨ Основной функционал

🗺️ **Интерактивная карта корпусов (Map Module)**
- Просмотр списка учебных корпусов с фотографиями (динамическая подгрузка и асинхронное декодирование).
- Маршрутизация в один клик: поддержка Apple Maps, Google Maps и Яндекс Карт (Yandex Navi).

📰 **Главный экран и новости (Home Module)**
- Красочная карусель (Hero Banner) с красивыми иллюстрациями и главными объявлениями на весь экран.
- Удобный доступ к полезным сервисам (электронная библиотека, образовательная среда, быстрый переход к нужным ссылкам через in-app Safari).
- Свежая лента новостей академии с подробными деталями.

📚 **Список заданий (Tasks Module)**
- Трекер дедлайнов и домашних заданий с умными индикаторами времени сдачи (зеленые — терпит, красные — срочно).
- Детальный экран для каждого задания с подробными описаниями, информацией о преподавателе и заметками.

## 🛠 Технологический стек

- **Язык разработки:** Swift 5.0
- **Фреймворк:** UIKit (100% программная верстка, никаких Storyboards или XIB)
- **Верстка:** [SnapKit](https://github.com/SnapKit/SnapKit) (надежный и читаемый DSL для Auto Layout)
- **Архитектура:** Модульный MVC подход (четкое разделение моделей данных, UI-компонентов и логики контроллеров)
- **Коллекции:** Современный подход с использованием `UICollectionViewCompositionalLayout` и `UICollectionViewDiffableDataSource`
- **Асинхронность:** Многопоточное скачивание и декодирование тяжелых фотографий через `DispatchQueue.global`

## 🚀 Сборка проекта

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-username/CampusNav.git
   ```

2. **Зависимости:**
   Откройте проект в Xcode. В проекте настроен Swift Package Manager (SPM), который автоматически подтянет зависимость ресурса `SnapKit`.

3. **Запуск:**
   - Откройте файл `CampusNav.xcodeproj` в **Xcode**.
   - Выберите нужный симулятор (или подключите реальное устройство с iOS 16.0 и выше).
   - Нажмите `Cmd + R` или кнопку **Run** ▶️.
